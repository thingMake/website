Html Documentation:

Each html tag needs to be closed. The closing tag is the same as the opening tag, except you insert a slash after the less then sign. Example: <p></p>

<!DOCTYPE html> This is not a html tag. It is just for telling the browser the document is written in html.
<html> Container for head and body elements.
<head> Container for metadata (data about data). Tags include: meta, style, link, title, etc; tags. Head element cannot be shown on page.
<body> This is where all the html goes. Can be styled with css.
<h1> --- <h6> These are headings. From heading 1 to heading 6.
<p> This is a paragraph.
<section> This is a section.
<a> This is a link. It needs 1 attribute which is "href". Example: <a href = "https://www.website.com"> text for link </a>
<title> This is where the title of the page goes.
